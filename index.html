JavaScript (discord.js) 入門

Node.jsとdiscord.jsライブラリを使ったBot作成の基本コード例です。

// index.js
const { Client, GatewayIntentBits } = require('discord.js');
require('dotenv').config();

const client = new Client({
  intents: [
    GatewayIntentBits.Guilds,
    GatewayIntentBits.GuildMessages,
    GatewayIntentBits.MessageContent
  ]
});

client.once('ready', () => {
  console.log(`Logged in as ${client.user.tag}`);
});

client.on('messageCreate', message => {
  if (message.author.bot) return;
  if (message.content === '!ping') {
    message.reply('Pong!');
  }
});

client.login(process.env.BOT_TOKEN);

Python (discord.py / py-cord) 入門

Pythonで作るDiscord Botのシンプルなサンプルコードです。

# bot.py
import os
from discord import Client
from dotenv import load_dotenv

load_dotenv()
TOKEN = os.getenv('BOT_TOKEN')

client = Client()

@client.event
async def on_ready():
    print(f'Logged in as {client.user}')

@client.event
async def on_message(message):
    if message.author.bot:
        return
    if message.content == '!ping':
        await message.channel.send('Pong!')

client.run(TOKEN)

デプロイ方法

Botをインターネット上で動かすには、以下のようなサービスを使います。

  • Replit（手軽で初心者向け）
  • Heroku（自動デプロイも可能）
  • VPSやクラウドサービス（GCP、AWSなど）

環境変数やAPIトークンの管理には注意しましょう。
